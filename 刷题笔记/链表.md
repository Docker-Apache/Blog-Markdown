## 链表的结构体

```c++
struct ListNode {
    int val;
    ListNode* next;
    ListNode():val(0), next(nullptr) {}
    ListNode(int x): val(x), next(nullptr) {}
};
```

## 创建链表

```c++
void LinkList_Create(ListNode*& head, vector<int>& nums) {
    if (head == NULL) {
        head = new ListNode;
    }

    ListNode* p = head;

    for (auto& num : nums) {
        ListNode* node = new ListNode(num);
        p->next = node;
        p = node;
    }
}
```
